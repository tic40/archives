javascript


	var app = angular.module("StudySessionApp", []);

	app.controller('GreetingController', function($scope) {
		$scope.greeting = 'Hello, world!';
	});
  • Controllers are where we define our apps behavior by defining functions and values
  • first line in javascript: declare the app
  • decalare the controller: controller is attached to inside our app(StudySessionApp)
  • $scope is the glue between application controller and the view.
  • in the html: declare ng-app in a tag, and declare ng-controller in a tag which is wrapped a tag declared ng-app
